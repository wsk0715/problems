## ✍️ 문제풀이 규칙
파일명은 `Solution_{문제번호}.java`와 같이 지정  
> 예: 재귀함수가 뭔가요?(`Solution_17478.java`)

개인적으로 느낀 중요도에 따라 파일명 앞에 언더바(`_`)로 강조 표시  
> 예: 토마토(`___Solution_7576.java`)

의무적으로 매일 n문제씩 풀지 않기. 원할 때 집중해서 풀기
